jasminejvm-maven-plugin

Run Jasmine from the command line.

Plugin Usage

configuration and defaults

        <plugin>
            <groupId>com.github.crazyfrozenpenguin</groupId>
            <artifactId>jasminejvm-maven-plugin</artifactId>
            <version>1.0.0</version>
            <configuration>
                <classpathScope>test</classpathScope>
                <pattern>target/**/*Spec.js</pattern>
                <format>DOC</format>
                <compileMode>OFF</compileMode>
            </configuration>
        </plugin>
  • Classpath Scope: system, compile, runtime or test. Defaults to test.
  • Spec files pattern: Defaults to "target/**/*Spec.js"
  • Format: DOC, PROGRESS. Defaults to DOC.
  • Compile Mode: OFF, FORCE, JIT. Defaults to OFF.

Tied by default to pre-integration-phase maven execution phase.

Goal: jasminejvm:run

Run Jasmine tests

    mvn jasminejvm:run
